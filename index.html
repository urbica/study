Воркшоп «Интерактивная картография и визуализация данных»

День 1: Основные принципы и инструменты

В первый день, вы познакомитесь с интерактивной картографией, научитесь работать картографическими данными и сможете их визуализировать. Для практических занятий мы подготовим несколько наборов данных, вы сможете выбрать.

Основные темы первого дня:

  • основные принципы представления информации
  • особенности восприятия картографической информации
  • типы визуализаций и разбор проектов на примерах
  • процесс разработки проекта по шагам
  • работа с данными: источники, форматы и способы обработки
  • знакомство с инструментами визуализации (CartoDB, Mapbox Studio)

День 2: Кодинг + индивидуальный проект Второй день в большей степени посвящён программированию интерактивных картографических проектов. Мы подробно рассмотрим несколько технологий. Во второй день, каждый участник будет работать над собственным проектом по визуализации данных (тему проекта, мы обсудим с каждым индивидуально до начала воркшопа). Познакомимся со следующими технологиями и фреймворками: Mapbox.js, Mapbox GL JS, Leaflet, D3.js, API Яндекс.Карт, CartoDB.js, PostgreSQL Для участвия в воршопе во второй день вам потребуются базовые знания веб-технологий HTML / CSS и JavaScript.
